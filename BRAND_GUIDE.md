# PROOFNEST Brand Guide

## Brand Essence

**Name Etymology:**
- PROOF = cryptographic verification, evidence, mathematical certainty
- NEST = origin point, birthplace, protected space where things begin
- Combined: "The birthplace of verification"

**Core Positioning:**
```
Bitcoin proved VALUE.
Proofnest proves EVERYTHING ELSE.
```

**Tagline:** "Where Proof is Born"

**Voice:** Confident, technical, trustworthy. Not hype, not corporate-speak.

## Logo System

### Symbol Mark
The Proofnest symbol represents:
1. A nest (circle/ring) - the protected origin
2. Three nodes - three types of proof (decision, timestamp, identity)
3. Connections - cryptographic links
4. Central core - the genesis of verification

### Logo Variations

1. **Full Logo** - Symbol + Wordmark (horizontal)
2. **Compact Logo** - Symbol + Wordmark (stacked)
3. **Symbol Only** - For favicons, app icons
4. **Wordmark Only** - Text-only contexts

### Clear Space
Minimum clear space = height of "P" in wordmark

### Minimum Sizes
- Full logo: 120px width
- Symbol only: 24px

## Voice & Tone

### We Are:
- Confident (not arrogant)
- Technical (not jargon-heavy)
- Direct (not salesy)
- Trustworthy (not corporate)

### We Say:
- "Proof, not promises"
- "Verifiable by anyone, anywhere, forever"
- "Stop trusting. Start verifying."

### We Don't Say:
- "Revolutionary" / "Game-changing"
- "Web3" / "Blockchain" (we say Bitcoin specifically)
- Vague claims without technical backing

## Usage Examples

### Good:
- "Quantum-proof signatures using Dilithium3"
- "Bitcoin-anchored via OpenTimestamps"
- "Cryptographic proof of who made each decision"

### Bad:
- "Blockchain-powered trust solution"
- "Next-gen verification platform"
- "AI-powered security"
